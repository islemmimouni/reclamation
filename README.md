## 📌 Description

Une application web de gestion des réclamations informatiques permettant aux utilisateurs de signaler des problèmes techniques et aux administrateurs d'assurer leur suivi et leur résolution.

---

## ✨ Fonctionnalités

### 👤 Utilisateur
- Créer une réclamation
- Suivre l’état des demandes
- Consulter l’historique

### 🛠️ Administrateur
- Gérer les réclamations
- Modifier les statuts
- Consulter les utilisateurs
- Tableau de bord complet

---

- HTML5
